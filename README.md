# ResearchMind AI

## Agent Debate Mode

This project supports a multi-agent research workflow in Streamlit.

### Debate flow
1. Literature Agent presents evidence.
2. Gap Agent challenges assumptions.
3. Hypothesis Agent proposes ideas.
4. Critic Agent attacks weaknesses.
5. Experiment Agent improves methodology.
6. Roadmap Agent creates the final plan.

### Current setup
- The main app is defined in app.py.
- The debate mode is displayed directly in the Streamlit interface.
- Future work will replace the placeholder debate text with real agent outputs and Azure AI Foundry integration.
